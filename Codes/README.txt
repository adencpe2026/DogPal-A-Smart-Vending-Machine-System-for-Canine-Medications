# DogPal Medicine Dispenser

## Overview
DogPal is a smart vending machine for canine medications that combines Raspberry Pi, Arduino, YOLO object detection, automated dispensing, and payment processing.

## Features

- YOLO-based medicine recognition
- Raspberry Pi GUI
- Arduino payment system
- Servo-controlled dispensing
- IR sensor drop detection
- Door security alarm
- Thermal receipt printing
- Service mode and maintenance logs

## Hardware Used

- Raspberry Pi 5
- Arduino Uno
- PCA9685 Servo Driver
- MPU6050
- IR Sensors
- Thermal Printer
- Coin Acceptor
- Bill Acceptor
